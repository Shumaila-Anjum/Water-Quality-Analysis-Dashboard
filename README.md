# 💧 Water Quality Analysis Dashboard

## 📌 Project Overview

This project was completed as part of the **Generation Australia Data Analytics Bootcamp** using **Microsoft Power BI**.

The objective was to analyse environmental water quality data to identify patterns, understand the impact of rainfall and land use on water quality, and evaluate whether pollution events affect fish populations.

The final deliverable is an interactive Power BI dashboard that supports data-driven environmental decision-making.

---

# 🎯 Objectives

The dashboard analyses:

- Impact of rainfall on turbidity
- Water quality across different land use types
- Nitrogen and phosphorus concentration
- Dissolved oxygen levels
- Fish population during pollution and non-pollution events
- Interactive filtering by year, quarter and land use

# 📂 Dataset

The dataset contains environmental monitoring data collected from multiple monitoring stations.

### Dataset Features

| Column | Description |
|---------|-------------|
| Station_ID | Monitoring station ID |
| Station_Location | Monitoring station location |
| Date | Observation date |
| Rainfall | Rainfall (mm) |
| Turbidity | Water turbidity (NTU) |
| pH | Water acidity/alkalinity |
| Dissolved_Oxygen | Dissolved Oxygen (mg/L) |
| Nitrogen | Nitrogen concentration |
| Phosphorus | Phosphorus concentration |
| Land_Use | Agricultural, Forest or Urban |
| Pollution_Event | Yes / No |
| Fish_Population | Estimated fish population |

# 📊 Key Insights

### Rainfall & Turbidity

Higher rainfall is associated with increased turbidity, suggesting runoff transports sediments and pollutants into waterways.

### Land Use

Agricultural areas generally show higher nutrient concentrations, indicating possible fertiliser runoff.

### Dissolved Oxygen

Dissolved oxygen levels vary across land use categories and may influence aquatic ecosystem health.

### Pollution Events

Fish populations differ between pollution and non-pollution events, highlighting the importance of ongoing environmental monitoring.

---

# 💡 Business Recommendations

- Increase monitoring after periods of heavy rainfall.
- Improve runoff management in agricultural areas.
- Continue monitoring nutrient concentrations.
- Investigate pollution events with declining fish populations.
